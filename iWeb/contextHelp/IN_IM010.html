<p>{0} : {1} imported as of {2}</p><p>{3} records have been added.</p><p>{4} records have been removed.</p>

Part of an information message that is sent by email to a Client upon  vivichart  importing a new Occurrence of an existing Data Source.

When generating the text within the email, vivichart will replace the following characters with other information. 

Characters Replaced with
{0} The name of the Data Source.
{1} The number of rows of data that have been imported.
{2} The Effective Date that has been applied to this data import.
{3} The number of new rows that have been imported. They are 'new' in that their Unique Reference did not exist in the prior Data Import of this Data Source.
{4} The number of rows that have been deleted. 'Deleted' in this context means that they existed within the prior Data Import of this Data Source but do not exist now.

Only the words that are shown in red below should be translated; the other characters and information in green are used by vivichart to format and populate the email.

<p>{0} : {1} imported as of {2}</p><p>{3} records have been added.</p><p>{4} records have been removed.</p> 

  Note Box
It is very important that the green characters are contained within the translated text and in the order that they appear within the English version.

This part of the email that is generated would look similar to:

Employees : 25 imported as of 25/03/2023

                     24 records have been added

                       5 records have been removed

You should use {} and not the () characters, within the translation.
