Unique Reference

A Unique Reference is the Field that is held within a Data Source that identifies each row of information and which an User has selected as being so.

 

In the following example, the 'Employee Reference' is intended to be the Unique Reference although 'Department' is unique currently. (There are two people called 'John Smith' so that is not uniqte within that Field.

Employee Reference Name Department
100 Andrew Black Administration
101 John Smith Sales
102 Jane Brown Marketing
103 John Smith Finance

 (The definition of a Unique Reference in each Data Source is mandatory).
