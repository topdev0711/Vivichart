The text of an email message that is sent automaticaly upon vivichar receiving and processing sussessfully a file attachment.

 

The {0}, {1}, {2}, {3}  and {4} characters are replaced by vivichart with:

  • {0} ??????????????????????????????????????????????????????? 
  • {1} the name of the file attachment,
  • {2} the time and date of when the file was imported
  • {3} the number of records that have been added
  • {4} the number of records that have been removed
You should use {} and not the () characters, within the translation.Type your Note Box content here.
