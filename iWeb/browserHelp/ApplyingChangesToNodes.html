Applying Changes To Nodes

The Attributes of a Node can be changed by

  • selecting the Node within a chart,
  • clicking the right mouse button, and
  • selecting the Edit option.

 

Changes made within the Details form that will be displayed can be made for the current Node only (by accepting the default Apply to this Node option) or cancelled.

 

The  Attributes of the current Data Node or a Group Node can be applied to other Nodes of the same type at various levels within a Chart.

 

If a Data Node is defined as being either an Assistant or Vacant, each of these will be deemed to be a separate type for this purpose.

 

So if a Vacant Node's Attributes are applied to other Nodes, this will only affect other Vacant Nodes.  Likewise, an Assistant Node's Attributes will only be applied to other Assistant Nodes. If a Data Node is both Vacant and an Assistant, all other Vacant Nodes and Assistants will assume that Node's Attributes (and NOT just those that are both Vacant and an Assistant).
