Displayed Text

Up to 6 lines of text can be displayed within any of the Nodes.

 

Text Source

This text will be either

  • determined by a Data View that has been used to display information within the Data Nodes of a chart,
  • copied from a Data Node when it is made Vacant, or
  • entered manually when a $Group Node$$ is added.

The text can be edited within each chart.

 

Vacant Nodes

A line of text within a Data Node that has been made $$Vacant may assigned a description that has been defined within the chart's corresponding Data View$$.

Display Attributes

The following is defined and can be changed for each line of text. 

  • Font
  • Font Size
  • Colour
  • Bold                
  • Italics              
  • Underlined      
  • Justification        Left       Centre      Right

 

Global Attributes 

Default Display Attributes for each of the 6 lines of text are defined for both Data Nodes and Group Nodes.

These settings can be modified and

  • made constant by choosing to not allow them to be overridden, or
  • changed within each chart.

 

Data View Attributes

If the Global Attributes have not been made constant, default Display Attributes for each $$Data View$$ can be set for each of the 6 lines of text that can be defined for both Data Nodes and Group Nodes. Upon creating a Data View, its Display Attributes will be set to those of the Global Attributes.

 

These settings can be modified and

  • made constant by choosing to not allow them to be overridden, or
  • changed within each chart.

 

SUMMARY 

The look of all lines of text that are displayed for Data Nodes and Data Groups can be made constant

  • across all charts,
  • some types of chart or
  • be different within each chart.
