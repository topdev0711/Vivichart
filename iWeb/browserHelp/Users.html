Getting Started > Users

There are two types of user.

 

Chart Builders who can create and publish Charts, and

Viewers who can view published Charts.

 

A Chart Builder may also be made a Administrator

 

A User ID, along with a Password, allow a person to log-into ViviChart.

 

User IDs

 

User IDs within vivichart are the same as persons' eMail addresses. The person who initiated the use of vivichart will have entered an eMail address and this eMail address will be the first User ID to be created. All subsequent User IDs must have the same Domain Name as the initial user.

 

 

 

 

 

Passwords

 

Password Policy

As standard vivichart requires that a password must have a minimum of 6 characters, of which 1 must be uppercase alphabetic, 4 lower case alphabetic and a digit (ie:  Z1exil )

Adding Users

A User can be added to vivichart from the User Maintenance Form. New Viewers can be added by a System Adminstrator sending an instructon by eMail to do so.

Retiring Users

A User's access to vivichart can be retired as of today's date or a date in the future from the User Maintenance Form or by a Administrator sending an Email with an instruction to do so.
