vivichart is designed to create its charts from information that is submitted to it from other sources.

 

All information that is imported will be associated with an effective date that allows for new charts to be created, whilst still retaining those that had been created previously.

Considerations

The information that is entered will be used to

  • define Selection Lists,
  • apply Filters to the Selection Lists so as to limit the information that is presented to individual users when they are creating charts,
  • populate the Data Nodes within charts, and
  • allow users to drill down to see further information about any Data Node.

 Selection Lists

You will want to display sufficient information within each row to identify its content accurately. If a Selection List  is displaying employees, 5 of whom have a surname of Jones, you will want to show more information on each line - Job Name, Department, Company, etc..

What you can import

You can import any number of the following types of information

  • Text
  • Numbers
  • Dates, and
  • Images

 

Images can be imported after you have uploaded other information. A specific process will import images.

 

 

What you must import

 

 

The following information is required to allow charts to be created from the Selection Lists. 

  • An unique name for each Data Source
  • The names of the Fields within each Data Source, and
  • The Data Type of each Field.

Each Data Source must also have either 

  • a single Field that contains Unique Information, or
  • one (or more) Field(s) that share the Unique Information of one (or more) other Data Source(s).
