Passwords

The entry of a Password is required to allow a User to access vivichart.

 

 

Password Controls

The minimum and maximum length of passwords can be set in the XXX form by Administrators, as can the minimum and maximum number of

  • Upper Case characters (A-Z)
  • Lower Case characters (a -z)
  • Numbers (0-9), and
  • other characters  that can be defined. (For example,  +*/ and !)

that must be included in each password, when it is changed by a User or generated as a temporary password for a new User.

 

Additional controls exist to limit the continued usage of vivichart  by its users.

Control Description
Passwords expire [0+] days after being set

All passwords can be set to expire a number of days from the date that they were created,

This control will not be used if 0 days is entered.. As a consequence, each user's usage of  will continue until an Administrator enters an expiry date for him/her.
Reminder issued [0+] days before expiry

Users will be reminded a number of days before their passwords are due to expire.

This control will not be used if 0 days is entered.
Suspend User After [0+] days inactivity

Users' usage of ViivChart will be suspended after a given number of days of inactivity.

This control will not be used if 0 days is entered.

Suspend User after [3-5] unsuccessful log-in attempts

 A User's usage of the service will be suspended after a given number of unsuccessful log-in attempts.
Suspend User if password expires Users' usage of vivichart  can be suspended automatically after the expiry of their passwords. A User who has had his/her usage suspended will require an Administrator to re-instate his usage, if required.  If usage is not suspended, users will be able to re-enter a new password after the expiry of his/her current password.
Bar reuse of earlier password Users can be required to create a new password that he/she has not used previously, when changing his/her password.

 

 

New Users

Each new User will be sent an initial and temporary password to his/her email address. Upon first logging into vivichart , the new user will receive a code by email that must be entered in order to proceed with his/her use of the service.
